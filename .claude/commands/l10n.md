# Localization Check

Check localization completeness for the Med Assist app.

1. Compare keys in `lib/l10n/app_en.arb` vs `lib/l10n/app_ar.arb` — list any keys missing from Arabic
2. Search Dart files for hardcoded user-facing strings (not in `.arb`) using grep for quoted English words inside widget build methods
3. Regenerate localizations:

```bash
flutter gen-l10n
```

Report:
- Missing AR keys (need translation)
- Suspected hardcoded strings with file:line locations
- Any gen-l10n errors
